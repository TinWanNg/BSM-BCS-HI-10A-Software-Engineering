# Pet Project — Lovable

A **virtual pet fitness app** generated by [Lovable](https://lovable.dev) — an AI-powered web app builder. It's the agentic coding section (task 13b) of the Software Engineering coursework, demonstrating AI-generated code via a GUI tool.

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 + TanStack Start (SSR) for pre-render |
| Routing | TanStack Router (file-based — all files in `src/routes/` automatically become a route, no manual config) |
| Data fetching | TanStack Query — handles loading, caching, and refetching |
| Storage | `localStorage` — all data persisted in the browser, no backend needed |
| UI | Radix UI + Tailwind CSS v4 (shadcn/ui style) |
| Build | Vite |

## Core Concept

A Tamagotchi-style health motivator:

1. **Track daily goals** — steps, water intake, sleep hours, weekly gym workouts
2. **Earn coins** — completing or exceeding a goal awards coins
3. **Spend coins in the shop** — buy food (boosts pet's *hunger* bar) or toys (boosts pet's *fun* bar)
4. **Pet welfare** — when hunger or fun drops below 30%, the pet alerts you to hit your goals

## Pages / Routes

| Route | Purpose |
|---|---|
| `/` | Redirects to `/home` |
| `/onboarding` | Name yourself and pick from 4 pets (`src/lib/pets.ts`) — shown on first visit |
| `/home` | Dashboard: pet avatar + goal progress bars + mood |
| `/log` | Log today's steps, water, sleep, workouts |
| `/goals` | Set your goal targets |
| `/shop` | Buy food/toys with coins |
| `/settings` | View profile, change pet, edit goals |

## Data & Storage

All app state is stored in `localStorage` via `src/lib/storage.ts`:

| Key | Contents |
|---|---|
| `petfit_profile` | Display name, pet choice, coin balance, onboarded flag |
| `petfit_goals` | Steps, water, sleep, and workout targets |
| `petfit_pet` | Hunger and fun levels (decay 4pts/hr when app is closed) |
| `petfit_logs` | Daily activity logs keyed by date |
