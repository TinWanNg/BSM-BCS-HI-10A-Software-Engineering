# Roadmap – Assistive Smart Control Platform

## Phase 1 – Core Platform (MVP)

* Accessible UI (keyboard navigation)
* Basic device control (lights on/off)
* User authentication
* API integration (of most common smart home API)

## Phase 2 – Communication Features

* Predefined message templates with customisation option
* Emergency alert functionality
* Contact management

## Phase 3 – System Enhancement

* Performance optimization
* Improved accessibility (voice input, UI improvements)
* Reliability improvements (error handling, retries)

## Phase 4 – Expansion (Future)

* Integration with more IoT devices
* Mobile support
* Advanced automation (custom commands, routines)
